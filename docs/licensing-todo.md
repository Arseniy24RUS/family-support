# Решения по лицензированию, требующие владельца проекта

Патч намеренно не добавляет лицензию автоматически: выбор условий должен учитывать права Института, информационного партнёра и происхождение собранных данных.

До публичного позиционирования проекта как открытой платформы следует отдельно определить лицензию программного кода, условия использования собственных текстов, режим повторного использования структурированных данных, основания воспроизведения материалов партнёра и правила цитирования выгрузок. Рекомендуется зафиксировать эти решения в `LICENSE`, `DATA-LICENSE.md` и публичном разделе методологии.
